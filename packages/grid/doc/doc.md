# Grid 宫格

## 概述

宫格组件（Grid）与 24 列栅格布局不同的是，栅格组件（Row、Col）可以自定义一行内的列数，及各列所占比例，宫格组件则是将每行等分为宽度相等的列，甚至可以将单元格设置为正方形，并支持动态适应屏幕，就像九宫格一样。
